Electricity consumption calculator privacy notice

Application does not store, analyze nor send your data anywhere. Its all localhost.

This tool uses Session Storage and Local Storage built within browser.
The Session and Local Storage are in other word the cookies.

Session Storage is a in-tab cookie storage method.
This way no cookie is stored longer than it should or needed to be.
This type of cookie is used for data itself during tool usage.
This cookie will be automatically deleted when tab is closed.

Local Storage is more persistent cookie storage than Session Storage.
That means upon closing browsing tab or browser, the cookie is not deleted.
This type of storage is used in case user needs autofilling function in this tool.
This tool uses this type of storage to store Network Charges value for user between tool usage sessions.
The cookie will expire in 90 days.
